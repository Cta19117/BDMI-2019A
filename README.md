## Requirements

软件要求：

- Python3
- Haskell
- Github
- Jupyter Notebook
- Tensorflow

课程要求：

- 有一定算法、数据结构基础
- 有微积分，线性代数，概率论基础
- 上课需要自带笔记本电脑


## 目录结构

- WWn/ 第n次课的课件和参考代码
- Study-Memo/ 同学们的课程小结（课程小结文件名格式：m-Dayn.md，m为学生编号，n为第n次课，学生编号在网络学堂的课程文件里可以找到)
- Course-Projects/ 课程实践任务
